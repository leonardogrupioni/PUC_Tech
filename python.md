# Ciclo 1 - Fundamentos de Python

**Por onde comecar?** Se voce so tiver tempo pra uma coisa, assiste o video do Guanabara sobre Python basico e ja escreve seu primeiro script no terminal. O resto voce absorve melhor durante e depois da aula.

## Videos (Portugues)

- **(DESTAQUE)**  Curso Python #01 - Seja um Programador (Curso em Video, Guanabara) - A referencia em portugues. Didatico, paciente, completo. Comeca por aqui.
- Curso de Python do Octavio Miranda - Cobertura densa do basico ao intermediario. Bom pra maratonar no fim de semana
- **Python para iniciantes - Hashtag Programacao** - Mais pratico, foca em resolver problemas reais rapidinho

## Videos (Ingles)

- **Python Crash Course - Traversy Media** - Vai direto ao ponto, tudo no terminal, em 1h
- **Learn Python - freeCodeCamp** - 4h cobrindo tudo do zero ao intermediario, bem estruturado

## Interativo

- **(DESTAQUE) Python Tutor** - Cola seu codigo e ve ele rodar passo a passo com visualizacao de variaveis na memoria. Indispensavel pra entender o que esta acontecendo de verdade
- **Exercism - Python Track** - Exercicios com feedback de mentores reais, do basico ao avancado. Comeca do nivel Hello World e vai evoluindo

## Artigos e Leitura

- **(DESTAQUE) Tutorial Oficial do Python (PT)** - Melhor que parece. O tutorial oficial e surpreendentemente acessivel e cobre tudo que voce vai ver na aula
- **Pense em Python (livro gratuito, PT)** - Traducao do classico "Think Python". Capitulos curtos, foca no raciocinio computacional
- **Real Python - Python Basics** - Artigos bem escritos e praticos, em ingles mas com exemplos muito claros

## Referencia Rapida

- **Python Cheat Sheet - gto76** - A cola mais completa que existe. Tem exemplo de tudo
- **W3Schools Python** - Consulta rapida de sintaxe com exemplos rodaveis direto no navegador

- Python e lento? A resposta honesta - Codigo Fonte TV

# Ciclo 2 - Orientacao a Objetos

**Por onde comecar?** OO e um jeito de pensar antes de ser uma sintaxe. Antes de codar, tenta desenhar no papel as classes, atributos e relacionamentos. Assiste o video do Otavio Miranda sobre POO e depois vai pro codigo.

## Videos (Portugues)

- **(DESTAQUE) POO com Python - Otavio Miranda** - Serie completa e bem cuidada. Cobre heranca, encapsulamento e polimorfismo com profundidade. A melhor serie de OO em portugues
- **Orientacao a Objetos em Python - Hashtag** - Mais rapido e direto, bom pra revisar depois da aula ou pra quem ja tem uma base
- **Os 4 pilares de OO em 20 minutos - DevAprender** - Foca na teoria dos 4 pilares sem codigo. Vale ver antes da aula pra chegar com o conceito na cabeca

## Videos (Ingles)

- **(DESTAQUE) Python OOP Tutorial - Corey Schafer** - Playlist de 6 videos. A referencia de OO em Python no YouTube em ingles. Muito didatico, ritmo otimo
- **OOP in Python - freeCodeCamp** - 2h cobrindo tudo de OO com Python do zero

## Artigos e Leitura

- **(DESTAQUE) Os 4 Pilares da OOP em Python - Medium PT** - Explica abstracao, encapsulamento, heranca e polimorfismo com exemplos em Python. Leitura rapida e essencial antes da aula
- **Python Classes - Real Python** - Artigo longo mas muito bem estruturado. Cobre desde `__init__` ate heranca multipla
- **Documentacao oficial - Classes** - Densa, mas tem tudo que voce vai precisar como referencia

## Interativo

- **Exercism - Classes em Python** - Filtra pelos exercicios com classes. Praticar e a unica forma de fixar OO de verdade
- **Python Tutor com Classes** - Visualize como os objetos sao criados e destruidos na memoria enquanto o codigo roda

## Referencia Rapida

- **Python Classes - W3Schools** - Consulta rapida de `__init__`, heranca, `super()` e metodos com exemplos prontos
- **Python Magic Methods - Cheat Sheet** - Referencia dos metodos especiais (`__str__`, `__len__`, `__eq__`...). Guarda esse link

## Conteudos Resenha

Vale refletir sobre quando usar e quando nao usar OO.

- OO e uma boa ideia? - perspectiva critica
- Por que todo mundo ensina OO errado - Akita
- Composicao vs Heranca - quando usar cada uma

# Ciclo 3 - Bibliotecas Python e IA na Pratica

**Por onde comecar?** Cria sua conta no Google Colab, abre um notebook novo e roda `print("hello world")` numa celula. A partir dai tudo fica mais facil. Depois cria sua conta na OpenAI e acessa o Playground pra brincar com o modelo antes da aula.

## Primeiro Passo

Antes de qualquer coisa, configura as duas coisas abaixo:

1. **Google Colab** - Precisa de conta Google. Cria um notebook novo e confirma que consegue rodar uma celula
2. **OpenAI Platform** - Cadastro gratuito com creditos iniciais pra experimentar. E a conta que vamos usar na aula pra chamar a API do ChatGPT

## Videos (Portugues)

- **(DESTAQUE) Google Colab do Zero - Teo Me Why** - Melhor introducao ao Colab em portugues. Cobre celulas, variaveis de ambiente e instalacao de libs
- **(DESTAQUE) API da OpenAI com Python na pratica - Filipe Deschamps** - Do zero ao chatbot funcionando em Python. Exatamente o que a aula vai construir
- **O que e pip e como instalar bibliotecas - Guanabara** - Entenda o ecossistema de libs Python antes de comecar a usar
- **Requests - fazendo chamadas HTTP com Python** - A lib `requests` em 20 minutos. Voce vai precisar dela pra entender como a API funciona por baixo

## Videos (Ingles)

- **OpenAI API Python Tutorial - freeCodeCamp** - Tutorial completo com a API da OpenAI em Python. Cobre autenticacao, chamadas, streaming e boas praticas
- **Python Libraries You Should Know - Tech With Tim** - Tour rapido pelas libs mais importantes do ecossistema Python

## Artigos e Leitura

- **(DESTAQUE) Quickstart da API da OpenAI (doc oficial)** - Em 10 minutos voce ja ta fazendo chamadas. Muito bem feito, le antes da aula
- **O que e um LLM? - explicacao acessivel** - Contexto importante pra entender o que voce vai estar usando. Leitura de 5 minutos
- **Python Requests - Real Python** - Guia completo da lib `requests`, em ingles mas muito didatico

## Interativo

- **(DESTAQUE) OpenAI Playground** - Experimenta a API direto no navegador antes de escrever uma linha de codigo. Otimo pra entender temperatura, tokens e como moldar o comportamento do modelo com system prompt

## Referencia Rapida

- **OpenAI Python SDK - GitHub** - Documentacao da biblioteca oficial. Guarda esse link pra consultar durante a aula
- **Awesome Python - libs por categoria** - Lista curada das melhores libs do ecossistema. Bom pra explorar depois da aula

## Conteudos Resenha

Pra entender o contexto maior do que voces vao estar construindo.

- Como a API do ChatGPT funciona por baixo dos panos
- O que sao tokens e por que eles importam
- Engenharia de Prompt em 10 minutos - Codigo Fonte TV
